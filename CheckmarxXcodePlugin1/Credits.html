The Plugin:

Welcome to the Beta!

--- Restrictions in the Beta ---

  • All Scans MUST be to a (server-side) bound Project
  • Private Scans are NOT supported
  • Only source files are zipped but there is NO include/exclude override
  • There is NO Proxy support
  • There is NO Data retention support

--- Items for GA ---

  • Ability to run an 'unbound' Project
  • Ability to create new Projects (Team\Preset\Engine Config\etc)
  • Include/Exclude override
  • Additional UI improvements (i.e. progress bars)
  • Add support for v9.0 authentication
